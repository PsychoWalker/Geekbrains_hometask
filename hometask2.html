1) Дан код. Почему код даёт именно такие результаты?

var a = 1, b = 1, c, d;
c = ++a; alert(c); // 2
d = b++; alert(d); // 1
c = (2+ ++a); alert(c); // 5
d = (2+ b++); alert(d); // 4
alert(a); // 3
alert(b); // 3

1. В alert(c) а сначала увеличивается до двух, а потом выводится.

2. В alert(d) постфиксная увеличивает на единицу, но возвращает прежнее значение

3. Далее мы увеличили а до трёх, а потом прибавили 2

4. В последней строке соответственно тоже есть увеличение, но вернётся прежнее значение, поэтому к b, который равен двум прибавляем 2.
